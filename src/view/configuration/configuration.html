Configuration
To setup your XDM development bridge we need a few pieces of information. For more information on how to utilize the extension, you can visit this page.

Data Object Name
The name of the data layer object attached to the window (ccsData).
Tenant Property Name
The top-level property name for your data, often based on your Adobe tenant ID (_bcbsnc).
Edit Schema
Define the allowed data structure of your data object. This schema applies to the data inside your Tenant Property.
Public Methods

Note: All paths are relative to your configured **Tenant Property Name**.

.get(path)

This method retrieves a single value from the data layer's state using dot-notation.

window.ccsData.get('user.authenticated');
.set(path, value)

Sets a value at a specific path in the data layer. Requires the path and value type to be defined in the schema.

window.ccsData.set('user.lob_plan_code', 'ABC-123');
.setView(viewName)

An alias for setPageName. This method sets the value of page.name.

window.ccsData.setView('Product Details');
.setAssets(path, assetsList)

Adds or updates assets within an `assets` array using a "list of pairs" format.

window.ccsData.setAssets('application', [['site', 'pubweb']]);
.merge(object, [path])

Merges an object into the data layer. If the optional path string is provided, the merge happens at that specific location.

window.ccsData.merge({
  form: { name: 'request-a-quote' },
  user: { ruid: 'some-random-user-id' }
});
Merge Shortcuts

The following methods are shortcuts for calling .merge(object, path) on specific top-level properties.

.setApplication(object)

Merges an object into the application property.

window.ccsData.setApplication({
  name: 'CustomerPortal',
  platform: 'desktop'
});
.setEvent(object)

Merges an object into the event property.

window.ccsData.setEvent({
  action: 'formStart'
});
.setForm(object)

Merges an object into the form property.

window.ccsData.setForm({
  name: 'newsletterSignup',
  source: 'footer-promo'
});
.setPage(object)

Merges an object into the page property.

window.ccsData.setPage({
  error_page: false,
  views: 3
});
.setUser(object)

Merges an object into the user property.

window.ccsData.setUser({
  authenticated: true,
  gender: 'female'
});
.setSearch(object)

Merges an object into the search property.

window.ccsData.setSearch({
  term: 'data layer',
  result_count: 15
});
Internal Methods
._getState()

Returns a complete copy of the entire data layer in it's current populated state.

window.ccsData._getState();
._getSchema()

Returns a copy of the JSON schema that the extension is using for validation.

window.cssData._getSchema();
